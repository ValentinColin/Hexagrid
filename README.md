Hexagons forever!

### Execute the app
```
cd Hexagrid
python3 main.py
```

### Control

| KEY        | description                                        |
| :--------- | :------------------------------------------------- |
| Arrow_up   | increase the size of the grid                      |
| Arrow_down | decrease the size of the grid                      |
| Space      | enable/disable the filling loop                    |
| r          | clear                                              |
| escape     | quit                                               |
| p          | print (in shell) the hexagrid's size               |
| s          | [DEBUG] (in shell) show the state of each hexacell |
